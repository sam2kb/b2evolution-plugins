reCAPTCHA Overview

A "captcha" is a program or test to diffrentiate between a human and a program. They are usually displayed as a garbled string of characters which humans can distuinguish easily, but computers cannot. A typical use of captchas on blogs are to prevent automated programs from filling up comment forms with unwanted advertisements.

The computer science department at Carnegie Mellon University designed the reCAPTCHA application to harness a human's superior character recognition capabilites in the field of book digitizing with the usefulness of traditional captchas. Their website, reCAPTCHA.net, provides more information.

To use reCAPTCHA, you must register with their site to recieve public and priviate keys for use on your website.

reCAPTCHA Installation

Download & Install

  1. The most recent version can be found at b2evolution forum.
  2. Unzip the file and copy or move the recaptcha_plugin folder to your b2evolution plugins folder.
  3. Log into as an admin to your backoffice
  4. Click on the App Settings tab, then the Plugins tab.
  5. The reCAPTCHA plugin should be visible under the "antispam" section of "Available Plugins". Use the [Install] action to install it.
  6. Done!

Plugin Configuration

This is a pretty straight forward plugin to configure. You will need the public and private keys you were assigned when you signed up with reCAPTCHA.

Plugin settings

  • Enter the Public and Private keys you recieved when you signed up with reCAPTCHA.net.
  • "Registration?" Check to enable reCAPTCHA on registration forms
  • "Comment?" Check to enable reCAPTCHA on comment forms
  • "Email?" Check to enable reCAPTCHA on email message forms
  • "Exempt Users?" Check to let registered (and logged-in) users submit comments and messages without solving a reCAPTCHA
  • "Enable SSL" Check _ONLY_ if your visitors are already interacting with your site over SSL already
  • "Apply Theme?" box if you want to apply a different colored theme/tab index/language to the reCAPTCHA box.
  • Choose your theme: red, white, blackglass, or clean
  • Select the tabindex for the reCAPTCHA box.
  • Select the language

Plugin variables (Advanced)

  • Code is set to "recaptch" by default; shouldn't need to be changed
  • Priority identifies the order in which plugins are called. Lower numbers priorites will be called before higher numbers.
  • Apply rendering? (Author's note: I'm not quite sure how this variable is supposed to interact with the plugin and b2evolution. On my installation, it is set to "stealth", and works for me. Future updates may clarify this setting.)
