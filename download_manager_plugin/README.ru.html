Download Manager plugin - Менеджер загрузок

Этот плагин позволит вам загружать файлы, управлять ими, вести детальную статистику загрузок, и добавлять ссылки на ваши файлы в записи блогов.

Плагин не открывает прямых ссылок на ваши файлы, что дает вам полный контроль над процессом загрузки. Вы сможете задать максимальную скорость загрузки как для зарегистрированных пользователей, так и для посетителей, временно отключать загрузку любого файла, разрешить загрузку файлов только зарегистрированным пользователям, а также многое другое.

Возможности плагина

  • Счетчик и детальная статистика загрузок.
  • Многопользовательский режим, собственные файлы и статистика для каждого пользователя.
  • Прямые ссылки на файлы не раскрываются, директории защищены при помощи .htaccess
  • Красивые ссылки на файлы типа www.yoursite.com/download/FILE_KEY
  • Защита файлов паролем.
  • Порог скачиваний файла, при превышении которого загрузка не возможна.
  • Загрузка файлов только для зарегистрированных пользователей.
  • Ограничение загрузки по уровню пользователя.
  • Ограничение скорости загрузки как для пользователей, так и для посетителей сайта.
  • Поддержка загрузки в несколько потоков и докачки файлов.
  • Правильная обработка запросов на прерывание загрузки, таким образом, загрузка файла прекращается мгновенно.
  • Полная поддержка менеджеров загрузки включая Reget, FDM, GetRight, FlashGet, DAP и т.д.
  • Разбиение файлов на группы/категории.
  • Ручная и автоматическая (через Cron) очистка статистики загрузок.
  • Экспорт статистики в виде .CSV файлов.
  • Кнопки в редакторе записей для добавления ссылок на файлы.
  • Уведомления на эл. почту пользователя о битых ссылках, утерянных файлах и др.
  • Изменяемая страница загрузок. Выводится непосредственно перед началом загрузки файла.
  • Полная поддержка нелатинских букв (Кириллица и др.) в именах файлов включая пробелы и специальные символы.
  • Поддержка локализации. В настоящее время доступен на русском и английском языках.
