MoreTags Overview

MoreTags is a tiny little toolbar that will let you add some common html markup tags to your blog post.

MoreTags Detailed Description

MoreTags is NOT a "WYSISYG" editor. For a real WYSIWYG editor check out TinyMCE. Using moretags will show you the markup in your blog post, much like the "quicktags" plugin. After installation moretags will create another toolbar under your "quicktags" toolbar that will operate in much the same way: you select the text you want to decorate and click the appropriate button.

  • The colorizing buttons provide spans tags for "rainbow" colors. I chose these colors (red, orange, yellow, green, blue, indigo, violet) because it was easier than trying to guess what will work best with your skin.
  • The big/small/underscore buttons provide span tags that will give you larger text or smaller text or underscored text. Not the most complete list of markup imaginable, but reasonable additions to the selection already available from Quicktags.
  • The heading size buttons provide h1 through h6 tags. If you also use the "Auto-P" plugin then you do not have to type in your "return return (heading text) return" to make it post correctly: auto-p and b2evolution will take care of making your post validate.
  • The "custom div" buttons provide divs that are customizable via the back office. The method to customize is not the easiest in the world to understand, but everything has to start somewhere. Scroll down a bit to the "settings" section for an explanation of what these do.
      NOTE: If you don't define your custom divs and use the buttons your post will look ugly because there will be a div with no meaning. Yuck!
  • You also get a special "X" button that will close any un-closed tags, not that b2evolution bloggers would ever leave an open tag laying around ;)

MoreTags Installation and Usage

    INSTALLATION
  1. Upload the "moretags_plugin" folder this file is part of to your "plugins" folder.
  2. In your admin panel, go to the "Global Settings->Plugins install" sub-tab.
  3. Click on "Install new" (right side, near top).
  4. Locate the More Tags plugin and click "Install" in the "actions" column.
    USAGE
  1. As you write your blog post select some text and click the button of your choice to get colors and stuff.

MoreTags Settings (with defaults)

This plugin has three settings with no default values. All three settings allow you to define custom div tags, and all three settings work the same way.

If you want to use a custom div you add ONLY the actual definition of the div. You do not need to add ".MT_div_1 {" then your definition then "}": only the definition part. For example, adding:

background-color:#87CEFA;margin:1em;border:1px solid #33A1DE;padding:1em

will automagically become

.MT_div_1 {background-color:#87CEFA;margin:1em;border:1px solid #33A1DE;padding:1em}

when you use the "div1" button

MoreTags Known Problems

  • The "custom div" buttons are sloppy. In a perfect world, the buttons wouldn't show up if you didn't define the div.
  • It's not easy to customize the entire toolbar. No chance to easily add your own colors, or "other style attributes" or your choice.
